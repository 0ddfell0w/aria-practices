Alert Example

This example shows a simple alert after you click the button. It’s implemented according to the design pattern for alert.

Example

Accessibility Features

  1. role=alert identifies an element as an alert which is a specialized form of the status role.

Keyboard Support

No keyboard interaction needed.

Role, Property, State, and Tabindex Attributes

Role Attribute Element Usage
alert Identify element as Alert widget
aria-live with implicit value of assertive Announces alert when it is displayed.
aria-atomic with implicit value of true Tells assistive technologies to announce all content.
