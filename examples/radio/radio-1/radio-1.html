Keyboard Support

Key Function
Tab
  • Moves keyboard focus to the checked radio button in a radiogroup.
  • If no radio button is checked, focus moves to the first radio button in the group.
Space
  • If the radio button with focus is unchecked, it's state will be changed to checked.
Down arrow
  • Moves focus to next radio button in the group.
  • If focus is on the last radio button in the group, move focus to the first radio button.
Up arrow
  • Moves focus to previous radio button in the group.
  • If focus is on the first radio button in the group, move focus to the last radio button.

ARIA Roles, Properties and States

Role Property/State Usage
radiogroup
  • Identify div element as a container for a group of radio buttons.
  • The descendent radio buttons are considered part of the group.
aria-labelledby
  • [IDREF]: Defines an accessible name (e.g. label) for the group of radio buttons.
radio
  • Identify div element a ARIA radio button.
aria-checked
  • [false]: Indicates the radio button is checked.
  • [true]: Indicates the radio button is unchecked.
  • CSS attribute selectors are used to synchornize the visual states with the value of the aria-checked attribute.
  • The CSS :before psuedo element with the content property is used to indcate visual state to support high contrast setting in operating systems and browsers.
tabindex
  • -1: All radio buttons in a group that are unchecked, except for the first radio button in the case when no radio buttons are checked.
  • 0: The radio button that is checked, or if no radio button is checked the first radio button in the group.
  • See roving tabindex for more information
