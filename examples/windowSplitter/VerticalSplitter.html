Keyboard Support

Key Function
Left Arrow (vertical splitter only) Move Splitter to the left
Right Arrow (vertical splitter only) Move Splitter to the right
Up Arrow (horizontal splitter only) Move Splitter to the up
Down Arrow (horizontal splitter only) Move Splitter to the down
Enter Minimize/Restore pane controlled by splitter

ARIA Roles, Properties and States

Role Property/State Component Usage
separator div
  • Identify element with the window splitter behavior
aria-expanded div
  • Indicates if the pane controlled by the splitter is expanded or collapsed
aria-controls div
  • Indicates which region is controlled by the splitter
aria-orientation div
  • Indicates if the splitter is a horizontal or vertical splitter
aria-label/aria-labelledby/title div
  • Provides a label for the splitter
