Layout Grid Examples

Note: This page is not ready for review; it is work in progress. When ready for review, feedback will be welcome in issue 105.

Following are three example implementations of the design pattern for grid that demonstrate how applying it can improve a keyboard interface while also providing semantic structure for a collection of widgets and associated descriptive information. In each example, a collection of widgets is grouped into a single tab stop by a grid widget that has a cell for each member of the collection and provides arrow key navigation among the members. While navigating the widget collection with the arrow keys, each member widget receives keyboard focus and is directly operable. If a screen reader is being used, since grid cells are containers that do not hide the semantics of their children, the screen reader can announce the role, state, and property information of the widgets that receive focus as well as the position of the widget within the collection.

Examples

Example 1: Simple List of Links

In this implementation:

  • Links can be focused using standard navigation keys, including directional arrows, Home, End, Control + Home, and Control + End.
  • The Page Down and Page Up keys are not supported since such scroll-like functions are not useful with so few rows.
  • The links can be activated with the keyboard when focused.

Example 2: Pill List For Components Like a List of Message Recipients

In this implementation:

  • The navigation keys and functions are the same as example 1.
  • Since text inputs require arrow keys to operate, the implementation and user experience are simplified by placing the input outside the grid.

Example 3: Scrollable Search Results

In this implementation:

  • In addition to the navigation keys supported in examples 1 and 2, Page Down and Page Up can scroll the grid.
  • As navigating up or down scrolls the visible set of rows, the aria-rowindex value of the row containing the focus informs the screen reader of the number of that row within the total set of rows.
